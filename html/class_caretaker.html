Detailed Description

Manages the stored Mementos, acting as a save-game manager.

  • The Caretaker is responsible for keeping the Memento objects safe, but it never operates on or examines the content of the Mementos. In this implementation, it manages a single "current" Memento and handles its persistence by saving and loading it from a file.
