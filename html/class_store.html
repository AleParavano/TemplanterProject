Detailed Description

Represents an in-game store where players can purchase items.

The Store holds a collection of available StoreItem objects and provides functionality for a Player to attempt to purchase an item.

Constructor & Destructor Documentation

◆ ~Store()

Destroys the Store object and cleans up memory.

It is assumed that the destructor is responsible for deleting the StoreItem objects stored in the 'items' vector to prevent memory leaks.

bool Store::purchaseItem ( size_t index,
Player * player )

Attempts to purchase an item from the store at a specific index.

The function checks if the player has enough currency and if the purchase is successful, it deducts the cost from the player and adds the item to the player's inventory.

Parameters
indexThe index of the item in the store's list (0-based).
playerA pointer to the Player attempting the purchase.
Returns
true if the purchase was successful, false otherwise (e.g., player lacks funds).
