Detailed Description

Represents the top-down, outdoor world map where buildings, NPCs, and traffic are rendered.

This scene is responsible for initializing the static world geometry, updating NPC movement, handling time of day, and managing transitions to indoor scenes (Store, Greenhouse, Inventory).
