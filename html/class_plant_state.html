Detailed Description

The Abstract State class in the State design pattern. It defines the interface for encapsulating the behavior associated with a particular state of the Plant.

  • This class manages the core metrics of a plant (growth, water, nutrients) and defines the virtual interface for state-specific operations like handling state transitions and ticking.
