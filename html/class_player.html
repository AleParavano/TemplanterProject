Detailed Description

The central class representing the player and managing the entire game state.

  • This class acts as the Originator in the Memento pattern, capable of creating a snapshot (Memento) of its internal state and restoring it from a Memento.
  • It handles finances, game time, worker management, and holds references to key components like Inventory and Greenhouse.
