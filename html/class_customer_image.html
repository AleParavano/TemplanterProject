Detailed Description

Flyweight class: Represents the Extrinsic state (shared data) of a customer, specifically the image (Texture2D) associated with a particular customer type.

This class acts as the Flyweight. Instances of this class are shared across many Customer objects of the same type, saving memory by loading and storing the image texture only once.
