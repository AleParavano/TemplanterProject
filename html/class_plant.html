Detailed Description

Represents a general plant entity in the game, managing its state, growth, and visualization strategy.

Forward declaration for the Context class (Plant).

  • This class implements the Context for the State pattern (using PlantState) and holds a reference to the Strategy pattern (PlantVisualStrategy) for its rendering logic.
