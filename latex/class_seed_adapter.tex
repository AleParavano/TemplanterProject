\doxysubsection{Detailed Description}
An Adapter class that makes a \doxylink{class_plant}{Plant} Factory function conform to the \doxylink{class_store_item}{Store\+Item} interface. 


\begin{DoxyItemize}
\item This is part of the {\bfseries{Adapter}} design pattern, allowing the store to sell seeds (which are \doxylink{class_plant}{Plant} Factory methods) as generic \doxylink{class_store_item}{Store\+Item} objects. 
\end{DoxyItemize}
