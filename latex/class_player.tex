\doxysubsection{Detailed Description}
The central class representing the player and managing the entire game state. 


\begin{DoxyItemize}
\item This class acts as the {\bfseries{Originator}} in the \doxylink{class_memento}{Memento} pattern, capable of creating a snapshot (\doxylink{class_memento}{Memento}) of its internal state and restoring it from a \doxylink{class_memento}{Memento}.
\item It handles finances, game time, worker management, and holds references to key components like \doxylink{class_inventory}{Inventory} and \doxylink{class_greenhouse}{Greenhouse}. 
\end{DoxyItemize}
