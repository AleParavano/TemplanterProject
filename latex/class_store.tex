\doxysubsection{Detailed Description}
Represents an in-\/game store where players can purchase items. 

The \doxylink{class_store}{Store} holds a collection of available \doxylink{class_store_item}{Store\+Item} objects and provides functionality for a \doxylink{class_player}{Player} to attempt to purchase an item.
