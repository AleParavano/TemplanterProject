\doxysubsection{Detailed Description}
Represents a general plant entity in the game, managing its state, growth, and visualization strategy. 

Forward declaration for the Context class (\doxylink{class_plant}{Plant}).


\begin{DoxyItemize}
\item This class implements the Context for the {\bfseries{State}} pattern (using \doxylink{class_plant_state}{Plant\+State}) and holds a reference to the {\bfseries{Strategy}} pattern (\doxylink{class_plant_visual_strategy}{Plant\+Visual\+Strategy}) for its rendering logic.
