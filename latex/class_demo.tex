\doxysubsection{Detailed Description}
The main application class that acts as the {\bfseries{Facade}} for the game demo. 

It initializes the raylib window, manages core subsystems (like the scene manager and player), and provides the main game loop ({\ttfamily \doxylink{class_demo_a508017df2cb4b7a099f6ddaf0b152d0f}{run()}}). It encapsulates the complexity of the subsystems.
