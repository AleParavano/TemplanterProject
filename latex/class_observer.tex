\doxysubsection{Detailed Description}
The \doxylink{class_observer}{Observer} interface (Abstract \doxylink{class_observer}{Observer}) defines an updating interface for objects that should be notified of changes in a \doxylink{class_subject}{Subject}. 


\begin{DoxyItemize}
\item This class is the abstract base for all concrete observer components (e.\+g., UI elements, logging systems) that need to react when the state of the \doxylink{class_greenhouse}{Greenhouse} (the \doxylink{class_subject}{Subject}) changes. 
\end{DoxyItemize}
