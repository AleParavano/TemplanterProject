\doxysubsection{Detailed Description}
An abstract base class that represents an object being observed (the "{}\+Publisher"{}). 

The \doxylink{class_subject}{Subject} maintains a list of dependents (Observers) and notifies them automatically of any state changes, usually by calling their update method.
