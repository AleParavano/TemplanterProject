\doxysubsection{Detailed Description}
The \doxylink{class_memento}{Memento} class stores the state of an Originator object (e.\+g., a game state) at a particular time, preventing direct access to the state data by other objects. 


\begin{DoxyItemize}
\item This class is part of the \doxylink{class_memento}{Memento} design pattern. It encapsulates data related to the game\textquotesingle{}s inventory, workers, greenhouse, finances, rating, and time. 
\end{DoxyItemize}
