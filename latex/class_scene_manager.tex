\doxysubsection{Detailed Description}
Manages the state machine for the game, controlling which scene is active, updating the active scene, and handling transitions between them. 

Implements the {\bfseries{State Pattern}} (or FSM) to switch between different game environments (Outdoor, \doxylink{class_greenhouse}{Greenhouse}, \doxylink{class_store}{Store}, etc.).
