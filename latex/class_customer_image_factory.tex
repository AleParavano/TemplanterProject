\doxysubsection{Detailed Description}
Singleton factory that manages and provides shared \doxylink{class_customer_image}{Customer\+Image} flyweights. 

Implements the {\bfseries{Flyweight Factory}} part of the pattern, ensuring that for any given customer type, only one {\ttfamily \doxylink{class_customer_image}{Customer\+Image}} object (flyweight) is created and reused.
