\doxysubsection{Detailed Description}
Manages the visual representation, rendering, and user interaction logic for the player\textquotesingle{}s inventory. 

This class acts as the {\bfseries{View/\+Controller}} component for the \doxylink{class_inventory}{Inventory} {\bfseries{Model}}, handling mouse input for slot selection, item swapping, and rendering the UI elements.
