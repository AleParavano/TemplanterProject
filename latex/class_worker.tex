\doxysubsection{Detailed Description}
An abstract base class representing an autonomous unit that executes commands. 

\doxylink{class_worker}{Worker} objects run on their own threads, process a queue of \doxylink{class_command}{Command} objects, and act as an \doxylink{class_observer}{Observer} to a \doxylink{class_greenhouse}{Greenhouse}, receiving updates when the \doxylink{class_greenhouse}{Greenhouse} state changes.
