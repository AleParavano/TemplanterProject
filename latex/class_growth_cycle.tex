\doxysubsection{Detailed Description}
The abstract base class for defining different strategies for plant growth. 

Forward declaration for the component responsible for managing plant growth.


\begin{DoxyItemize}
\item Implements the Strategy design pattern, allowing a \doxylink{class_plant}{Plant}\textquotesingle{}s growth mechanism to be swapped. 
\end{DoxyItemize}
