\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Project Overview}{Project Overview}}\label{md__r_e_a_d_m_e_autotoc_md2}
{\bfseries{Templanter}} is a plant nursery business management simulator built in C++ using the raylib graphics library. Players manage a virtual greenhouse, cultivate various plant species, hire workers, and run a profitable business by selling harvested plants to customers.

The project demonstrates the implementation of {\bfseries{11 design patterns}} in a practical, real-\/world application\+:
\begin{DoxyItemize}
\item {\bfseries{Singleton}} (\doxylink{class_game}{Game} state management)
\item {\bfseries{Factory}} (\doxylink{class_plant}{Plant} and \doxylink{class_customer}{Customer} creation)
\item {\bfseries{Adapter}} (\doxylink{class_plant}{Plant} to \doxylink{class_store}{Store} integration)
\item {\bfseries{Facade}} (System simplification)
\item {\bfseries{Flyweight}} (Data sharing optimization)
\item {\bfseries{\doxylink{class_command}{Command}}} (\doxylink{class_worker}{Worker} task execution)
\item {\bfseries{\doxylink{class_observer}{Observer}}} (\doxylink{class_plant}{Plant} monitoring)
\item {\bfseries{State}} (\doxylink{class_plant}{Plant} lifecycle)
\item {\bfseries{Strategy}} (Rendering algorithms)
\item {\bfseries{Template Method}} (Growth algorithms)
\item {\bfseries{\doxylink{class_memento}{Memento}}} (Save/\+Load system)
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysubsection{\texorpdfstring{Key Features}{Key Features}}\label{md__r_e_a_d_m_e_autotoc_md3}

\begin{DoxyItemize}
\item {\bfseries{10 \doxylink{class_plant}{Plant} Species}} with unique growth rates and prices
\item {\bfseries{4 \doxylink{class_plant}{Plant} Life States}} (Seed → Growing → Ripe → Decaying)
\item {\bfseries{3 \doxylink{class_worker}{Worker} Types}} (Water, Fertilizer, Harvest) with automated task queueing
\item {\bfseries{25-\/\doxylink{struct_slot}{Slot} \doxylink{class_inventory}{Inventory}}} with plant stacking support
\item {\bfseries{128-\/Plot \doxylink{class_greenhouse}{Greenhouse}}} with real-\/time plant management
\item {\bfseries{Complete Save/\+Load System}} for game state persistence
\item {\bfseries{Threading}} for worker operations
\item {\bfseries{Day/\+Night Cycle}} with time acceleration
\item {\bfseries{\doxylink{class_customer}{Customer} System}} for selling harvested plants
\end{DoxyItemize}
